# ⭐ 포트폴리오용 프로젝트 PRD
**"삼양 트렌드·크리에이터 인사이트 AI 에이전트"**

(popow.ai 제출용)

---

## 0. 프로젝트 요약

삼양식품의 글로벌 SNS 운영(틱톡·릴스 중심)에서 필요한 핵심 업무는
① 트렌드 탐지, ② 바이럴 포맷 분석, ③ 적합한 크리에이터 매칭이다.

본 프로젝트는 popow.ai를 통해 실행 가능한 AI 에이전트 1개를 설계해,
위 3가지 업무를 자동화·시각화하여 DXT팀이 바로 사용할 수 있는 형태의 프로토타입을 제공한다.

**에이전트 이름**: Samyang Viral Insight Agent

**핵심 기능**:
- 최신 틱톡/릴스 트렌드를 수집해 삼양 브랜드 관점으로 분석
- "삼양에 맞는 밈 포맷"과 "오늘 당장 실행 가능한 숏폼 아이디어" 도출
- 캠페인에 적합한 크리에이터를 자동 추천
- 전체 리포트를 popow.ai 인터페이스에서 즉시 확인 가능

---

## 1. 문제 정의 (Pain Points)

DXT 팀은 JD에서 다음과 같은 업무를 명시하고 있다:
- 틱톡 트렌드 분석
- 바이럴 영상 행동 분석 → 최적화 아이디어 도출
- 크리에이터 온보딩 심사/매칭
- 실험 중심의 업무 자동화 문화

이 업무들은 다음의 문제를 가지고 있다.

### 문제 1. 트렌드 변화 속도가 너무 빠름
- 매일 숏폼 플랫폼을 직접 모니터링해야 함
- 브랜드와 상관없는 트렌드가 많아 "노이즈 제거"가 필요함

### 문제 2. 바이럴 포맷 분석에 시간이 많이 들음
- 시각적 패턴, 음악, 텍스트 레이아웃, 편집 리듬 등
- 사람이 눈으로 분석하면 1개 영상당 최소 3~5분 소요

### 문제 3. 크리에이터 매칭 기준이 일관적이지 않음
- 조회수/팔로워 외에 "삼양과 잘 맞는 톤앤매너인지" 판단이 필요
- 매일 크리에이터 풀을 평가해서 A/B 테스트에 쓸 수 있는 인력을 확보해야 함

### 문제 4. 반복적 판단 & 리포트 생성 업무가 많음
- 매일 트렌드 요약
- 캠페인용 아이디어 제안
- 크리에이터 후보 리스트 생성

## 4. 핵심 기능 요구사항 (Key Features)

### ✔ 기능 1. 트렌드 수집 & 분석

**입력**
- 키워드 (예: "라면", "매운 음식", "K-food")
- 언어/국가 선택 (한국/미국/일본 등)

**처리**
- TikTok / Instagram Reels / YouTube Shorts
  (실제 API 없으면 검색 기반 정렬 + 분석용 텍스트 데이터 생성 방식 사용)
- LLM이 다음 요소 분석:
  - 포맷 유형(POV, Reaction, Meme, Transition 등)
  - 훅 문장 구조
  - 시각적 패턴
  - 음악/효과음 패턴
  - 댓글 반응 & 참여 행동 유도 방식
  - 삼양과 연결 가능한 포인트

**출력**
- "오늘의 삼양형 트렌드 Top 5"
- 각 트렌드에 대해:
  - 바이럴 포맷 요약
  - 삼양식품 적용 사례
  - 예상 성과 (정성 모델링)

### ✔ 기능 2. "삼양형 숏폼 포맷" 자동 아이디어 생성

**입력**
- 삼양 브랜드 카테고리 선택 (불닭, 삼양라면, 젤리 등)
- 원하는 톤앤매너 (재미/카와이/도발적/쿨톤 등)

**처리**
- 트렌드 분석 결과 + 삼양 브랜드 캐릭터 세계관을 LLM이 결합
- 영상 구성안 3개 제공:
  - 5초 훅 문장
  - 장면 구성 3~5컷
  - 편집 포맷
  - 추천 음악 스타일
  - 필수 소품/배경 아이디어

**출력 예시**
- **포맷 이름**: "Spicy POV Transformation"
- **5초 훅**: "불닭 처음 먹은 미국 친구의 반응은…?"
- **컷 구성** + 촬영 가이드

### ✔ 기능 3. 크리에이터 자동 매칭 (온보딩 심사 에이전트)

**입력**
- 크리에이터 SNS 링크
- 캠페인 목적 (인지/참여/전환)
- 타깃 국가

**처리**
- LLM 기반 프로필 분석:
  - 콘텐츠 톤앤매너
  - 과거 브랜드 협업 패턴
  - 매운 음식/식품 관련 콘텐츠 비중
  - 삼양 브랜드와의 적합도 점수 산정(0~100)
  - 리스크 요인 분석

**출력**
- "오늘의 추천 크리에이터 Top 3"
- 점수표 + 설명 + 제휴 아이디어

### ✔ 기능 4. 리포트 자동 생성

**처리**
- 위 3개 기능의 결과를 조합해:
  - 데일리 트렌드 리포트(PDF-like 텍스트 블록)
  - 크리에이터 매칭 리포트
  - "바로 촬영 가능한 콘텐츠 아이디어" 문서

**출력**
- popow.ai 대화창에서 리치 텍스트 카드 형태로 표시
- JSON 형태로도 출력 가능해 자동화 연결을 상정

## 5. 기본 동작 흐름 (Workflow)

```
[사용자 입력]
   ↓
[트렌드 수집 모듈]
   ↓
[LLM 분석: 바이럴 요소 추출]
   ↓
[LLM Reasoning: 삼양 브랜드 맵핑]
   ↓
[출력 ①: 오늘의 트렌드 인사이트]
   ↓
[사용자: 특정 트렌드를 선택]
   ↓
[LLM: 삼양형 숏폼 아이디어 생성]
   ↓
[사용자: 크리에이터 매칭 요청]
   ↓
[크리에이터 프로필 분석]
   ↓
[출력 ②: 추천 크리에이터 + 협업 아이디어]
```

## 6. 기술 스택

### popow.ai 내 구현 요소
- **LLM**: GPT-4.1 / Claude 선택 가능
- **Tools**:
  - Web search tool
  - Fetch tool for 프로필 페이지 크롤링
  - JSON output formatting
  - Multi-step reasoning prompt
  - Memory OFF (일관된 결과 보장)

### 외부 API
(포트폴리오 제출용이라 꼭 쓰지 않아도 되지만 있으면 플러스)
- YouTube Data API
- TikTok 검색 API 대체 수단 (구글 검색 + 요약 방식)

## 7. 프롬프트 아키텍처 설계

### System Prompt 핵심 요소
- 삼양 브랜드 정보를 요약해 정의
- 분석 기준(편집 패턴, CTA, 음악 패턴 등) 고정 규칙화
- 출력 형식(JSON + Human readable)
- "전문 숏폼 전략가 Persona" 지정
- 밈/포맷 분류 기준 테이블화

### Agent Prompt 예시(요약)
- "수집된 데이터에서 노이즈를 제거하고, 삼양 브랜드와 연결될 수 있는 요소만 추출하라."
- "각 트렌드의 바이럴 포인트를 최소 3개 이상 명시하라."
- "삼양이 실제로 사용할 수 있는 촬영 아이디어를 현실적인 수준으로 제안하라."

## 8. 성공 기준 (Success Metrics)

### 정성 기준
- popow.ai 대화창에서 1분 내에 실행 가능한 완성도 있는 인사이트 출력
- 삼양 브랜드를 "이해한 것처럼" 보이는 Reasoning 품질
- 콘텐츠 기획자 입장에서 바로 사용할 수 있을 정도의 구체성

### 정량 기준
- 입력 → 분석 결과 출력까지 20초 이내
- 5개 이상의 촬영 아이디어 생성
- 크리에이터 적합도 평가 요소 4개 이상 활용
- 오류 없는 JSON 정렬 출력 ≥ 95%
